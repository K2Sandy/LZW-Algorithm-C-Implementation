# LZW-Algorithm-C-Implementation
Implementasi sederhana algoritma kompresi dan dekompresi LZW (Lempel–Ziv–Welch) menggunakan bahasa C. Mendukung kamus 12-bit, operasi I/O berbasis bit, serta tool CLI untuk mengompresi dan mendekompresi file. Repository ini dibuat untuk tugas kelas Pemrograman B tahun ajaran 2025/2026 di bawah dosen pengampu Mahyus Ihsan, S.Si, M.Si.
